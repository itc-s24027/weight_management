# 体重管理機能

1 年生の Java の授業で作成したローカル環境で動く機能です。

## 概要
- Javaの基本を学んだ後の課題で作成
- ユーザー管理のできる体重管理機能
- 体重を記録し、目標体重を目指すシンプルな機能
- ターミナル上で操作できます

`⚠注意`
体重管理はユーザーが長期間使ってくれることが前提ですが、この授業ではデータベースを使用しておらず、メモリ上でデータを保存するため、実行終了すると登録した情報はリセットされてしまいます。

### ターゲット

- 女子生徒・最近太り気味の方
- 自分でも使えるような実用的なものを作ってみたかった

### 機能

- ユーザー登録
- ログイン・ログアウト機能
- 体重の記録
- BMI 計算
- ユーザー情報の変更


### ファイル構造
- Main.java
