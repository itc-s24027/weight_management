# 体重管理アプリ

1 年生の Java の授業で作成したローカル環境で動くアプリです。

## 概要

- ユーザー管理のできる体重管理アプリ
- 体重を記録し、目標体重を目指すシンプルな
- ターミナル上で操作

`⚠注意`
このアプリはユーザーが長期間使ってくれることを前提につくっていますが、この授業ではデータベースを使用しておらず、メモリ上でデータを保存するため、実行終了すると登録した情報はリセットされてしまいます。

### 機能

- ログイン・ログアウト機能
- 体重の記録
- BMI 計算
- ユーザー情報の変更

### ターゲット

- 女子生徒・最近太り気味の方
- 自分でも使えるような実用的なものを作ってみたかった思いもあります

## 使用技術

- Java
